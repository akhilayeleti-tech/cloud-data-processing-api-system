# Cloud Data Processing & API System

## Overview
Backend system developed for processing and handling data using REST APIs and cloud technologies.

## Features
- REST API development
- Data processing workflows
- Cloud deployment on AWS
- Database integration
- Docker containerization

## Technologies Used
- Java
- Python
- MySQL
- MongoDB
- AWS EC2
- AWS S3
- Docker

## Purpose
Designed to process and manage cloud-based data workflows efficiently using scalable backend architecture.
